= GitHub CLI

Install Github CLI.

The arguments (`devon gh «args»`) are explained in the following table:

.Usage of `devon gh`
[options="header"]
|=======================
|*Argument(s)*    |*Meaning*
|`setup`          |install GitHub CLI on your machine.
|`«args»`         |call GitHub CLI with the specified arguments.
|=======================
